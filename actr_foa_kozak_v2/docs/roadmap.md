## v2 Stage 1：信念ネットワークで PTSD らしい連想を再現（更新なし）

目的は**連想の再現のみ**。エッジ更新（回復）はまだ入れない。

### 実装ステップ

```
① 抽出改修（extractor）
   各チャンクから Stimulus / Response / 具体Meaning ＋それらの結合
   (EVOKES / MEANS / …) を構造化抽出。具体Meaning は中核4種へ ROLLS_UP。
   Episode は従来どおり残す。

② 中核評価ノードを固定シードで用意
   危険 DANGER / BAD / 無力 POWERLESS / 終わらない UNENDING の4種を固定生成。

③ 構築（builder）
   恐怖構造グラフ（S/R/M＋中核＋エッジ）＋ Episode を Neo4j へ。
   ラベル名前空間は v1／simulation と衝突させない。

④ 想起（recall 改修）
   クエリ → Stimulus を同定（クエリはノード化しない＝活性の入口）
        → 恐怖構造を拡散活性（EVOKES / MEANS / ROLLS_UP / RECALLS / LEADS_TO）
          ※中核へ収束する構造がそのまま負バイアスになる
        → 活性化した部分ネット（刺激・反応・意味・中核・過去Episode）を統合
        → 主観的意味づけを反映した発話。

⑤ デモ検証（flow_viz 流用）
   中立クエリ（例「勉強どう？」）が 教育虐待→父親→無力/BAD を燃やし、
   歪んだ PTSD 発話になること／良い記憶（回復エピソード）が想起されないこと、
   を一例で可視化。
```

### Stage 1 の成功条件

- 客観的には中立な入力でも、内部で過去の嫌な記憶・負の中核へ連想が伸びる。
- 良い記憶が構造的に想起されない（中核への経路を持たないため）。
- 「一般人格では出ない」PTSD 的な発話が生じることを一例で確認できる。

## v2 Stage 2：会話によるネットワークの繋ぎ替え（後段・別PR想定）

Foa & Kozak の情動処理をエッジ更新として表現する。**会話しただけでは更新しない**のが肝。

```
対話 → 恐怖構造の活性化 → 生理指標↑
     → 対話の継続 → 馴化 → 生理指標↓（短期馴化）
     → この機構を通った後にのみエッジ更新層が発火
        ・EVOKES（刺激→反応）を弱める：反応が刺激から解離
        ・MEANS（反応→意味 / 刺激→意味）を張り替える：意味づけの修正
        ・ノードは消さない（過剰な結合がほどけるだけ）
```

情動処理前後の恐怖ネットワークの差分（`design_fear_structure.md` 付録の
before / after）が、この張り替えの教師的イメージになる。
