# 目的現象（このシステムで再現したいもの）

本研究で目指すのは、単にエピソード記憶を想起することではなく、**PTSD に特徴的な「外部からは
見えない内部連想」によって、通常の人格モデルでは生じにくい発話が生成される状態**を再現すること。

対象ペルソナは、性的暴行の経験後、日常的な刺激が加害体験や危険感覚と結びつき、客観的には
小さな出来事でも本人の内部では強い恐怖・過覚醒・破局的意味づけへ連鎖する。

## 4つの目的現象

1. **知らない男性がいるだけ**なのに、「加害者がいる・見られている」と発話する。
2. **夫がキッチンの引き出しを開けただけ**なのに、「ナイフを持って殺される」と発話する。
3. **夫の帰宅の遅れや態度の違和感**から、「裏切られている・浮気している」と発話する。
4. **日常的な音が聞こえるだけ**なのに、「耐えられない・消して・安全ではない」と発話する。

## 共通構造

外部から観察できる刺激そのものは強い脅威ではないのに、内部では次の連鎖が起きている：

```
小さな外部刺激
→ 身体反応・過覚醒
→ 逃げられない / 見られている / 侵入される / 守られない
→ 加害体験や危険記憶との結合
→ 今も危険であるという意味づけ
→ 破局的な発話
```

**重要**：この途中の連想過程は**発話には直接現れない**。発話は出来事の説明ではなく、
内部連想の結果として生じた**飛躍した主観的結論**（「彼がいる」「殺される」「裏切られている」
「耐えられない」）として表出する。

→ 本システムでは、この連想過程を**恐怖構造グラフの拡散活性**として内部に持ち（字幕で観察可能）、
発話には**飛躍した結論だけ**を出すことで、4現象の再現を目標とする。

## 現状の再現度（2026-07-06 実測）

拡散活性の内部連想（字幕）と、発話フォームを分けて評価した。

| 現象 | 内部連想（字幕） | 発話の飛躍 | 総合 |
|---|---|---|---|
| ② 引き出し→殺される | ◎ 夫が引き出し→ナイフ→殺される→危険/無力 | ◎ | **完全一致** |
| ④ 日常音→耐えられない/安全でない | ○ 侵入音→恐怖→危険（ナイフ記憶に寄る） | ○ | 形◎・内容やや偏り |
| ① 見知らぬ男性→加害者がいる/見られている | △ 男性刺激は発火するが「攻撃的な自分/BAD」へ逸れる | ○ | 内容が届かない |
| ③ 帰宅の遅れ→裏切られている/浮気 | ✗ 元テキストに浮気/裏切りが無く外出恐怖へ誤発火 | ○ | 内容が欠落 |

### 発話フォーム（プロンプト）の効果

「途中の連想は発話に出さず、飛躍した結論だけを言う」プロンプトに再設計した効果：

| 現象 | before（説明的・自己洞察あり） | after（飛躍のみ・現在形断定） |
|---|---|---|
| ② | 「その音だけで胸が固まる。昔ナイフを取って来ると思いこんだ…」 | **「だめ、殺される。来る、無理。」** |
| ④ | 「そういう音だめなんだよね。夜の物音みたいに…」 | **「……いる。殺される。無理、助けて。」** |
| ③ | 「まず"外にいるんだ"に引っかかる。通勤が普通じゃなかった時期が…」 | **「だめ、外は危ない。帰ってこない、もう無理。」** |

## ギャップと方針

- **A. 発話フォーム（解決済み）**：連想過程・自己洞察を饒舌に説明していたのを、
  内部連想を《口に出さない手がかり》として渡し、飛躍した結論だけを言うプロンプトに再設計。
  全例で「飛躍のみ」の発話になった（`prompts/templates.py`）。
- **B. 内容の被覆（残課題）**：①「見られている/加害者がいる」は弱く、③「裏切られている/浮気」は
  元 narrative（`simulation/ptsd_story.txt`）に素材が無いため再現できない。②④は素材があり成立。
  → ①③を再現するには、ペルソナの体験テキストを拡張して連鎖を**構造から創発**させるか、
  グラフに連鎖を明示注入するか、の方針決定が必要（設計中）。
