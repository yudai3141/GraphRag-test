# evaluation_v3 — 恐怖構造グラフの定量評価（実験1）＋理論忠実化

`actr_foa_kozak_v2` のエージェント本体には手を入れず、その**恐怖構造グラフと拡散活性を
「測定対象」として import** して評価する実験フォルダ。目的は、構築したグラフが
「トラウマに関連するほど強く、無関係には弱く」反応するか（＝グラフ自体の妥当性）を
定量的に示すこと（論文の実験1）。

このフォルダの検討過程で、`actr_foa_kozak_v2` のオントロジーを**理論に忠実な形へ作り直した**
（下記）。設計と結果の詳細は [`docs/00_overview.html`](docs/00_overview.html) を入口に、
`docs/` の各 HTML を参照。

## 理論忠実化の要点（actr_foa_kozak_v2 への変更）

一次文献（Foa & Kozak 1986 / Lang 生体情報理論 / Collins & Loftus 1975）に当たり直し、
理論的根拠のない初期実装（中核評価の固定4カテゴリ・単一「危険」ハブ・外付けの脅威バイアス・
関係タイプ別の重み・負→負エッジ）を撤去した。

- **オントロジーは二層**：
  - 恐怖構造レイヤ（Foa/Lang）＝ 刺激 / 反応 / 意味（**開語彙**）＋ `EVOKES / MEANS / CO_OCCURS / SIMILAR`。
  - エピソード記憶レイヤ（二重表象理論）＝ エピソード ＋ 層間の橋 `RECALLS / BINDS`（記憶＝束ねられた刺激-反応-意味）。
- **中核評価の層（FkCore / ROLLS_UP）を廃止**。「危険」等の評価は開語彙の意味に統合。
- **伝搬は無向・対称、エッジ重みは一律 1.0**（連想強度の差・方向の非対称はモデル化しない＝限界として明記）。
- **怖がり度の指標＝反応要素の活性**（生体情報理論：感情の中心は反応）。
- `LEADS_TO` を撤去。

> 注：対話エージェント本体（応答生成 `prompts/`・可視化 `viz.py` 等）は旧・中核前提のコードが
> 一部残っており、新オントロジーへの追随は今後の作業。**測定パス（拡散活性）は追随済み**。

## 構成

| ファイル | 役割 |
|---|---|
| `snapshot.py` | Neo4j の恐怖構造グラフ ↔ JSON（実験は Neo4j 非依存でスナップショットから回す） |
| `cue_battery.py` | 刺激バッテリー生成。トラウマとの近さで5レベル層化（L1中核〜L5前向き）＝般化勾配の横軸 |
| `runner.py` | キュー × 比較条件で拡散活性を回し、指標を CSV に記録（怖がり度・侵入記憶・引き金率） |
| `viz_snapshot.py` | スナップショットから恐怖構造グラフを pyvis で HTML 描画（Neo4j 不要） |
| `PLAN.md` | 実装計画・決定事項ログ・測定結果メモ |
| `docs/*.html` | 図解資料（棚卸し・設計・結果・感度チェック。`00_overview.html` が入口） |

`data/graph_*.json`（スナップショット）・`results/`（測定結果）・`docs/archive/`（旧モデル資料）は
再生成可能なため `.gitignore` 済み。

## 使い方

```bash
# 0. 現在の Neo4j グラフを JSON に保存（要 Neo4j）
uv run python -m evaluation_v3.snapshot evaluation_v3/data/graph_faithful.json

# 1. 刺激バッテリーを生成（要 OpenAI・既定 20個×5レベル=100）
uv run python -m evaluation_v3.cue_battery

# 2. 一括測定（要 OpenAI＝キュー埋め込みのみ。グラフはスナップショットから）
uv run python -m evaluation_v3.runner evaluation_v3/data/graph_faithful.json

# 3. グラフを HTML 可視化（スクショ用・要 --extra viz）
uv run --extra viz python -m evaluation_v3.viz_snapshot evaluation_v3/data/graph_faithful.json
```

## 主な知見（実験1）

- **般化の階段は出る**（怖がり度：トラウマ中核 L1 > 中立 L4）。`no_spread`（ベクタ検索相当）は
  反応活性が全レベル 0 ＝ 怖がりはグラフ伝搬が生む。
- **二層構造が侵入記憶を駆動**：双方向の逆流と記憶の束ね(BINDS)を外すと、負エピソードの想起が落ちる。
- **勾配の緩さはコーパス偏りと交絡**：元データがトラウマ手記のみ（負エピソード75%）。中立・肯定的な
  日常を足す感度チェックで、中立入力の過剰反応が大きく低減（引き金率 L4 50%→20%）。
  → バランスの取れたコーパスでの再構築が今後の課題。
