実験2の設計 — 2026年7月15日

CAPS-5 の症状を、
グラフの“どこ”で測るか

実験2は、発話を加えて「PTSD 患者らしさ」を CAPS-5（診断用の構造化面接）の B 項目に沿って再現・評価する。その中核は、同じ拡散活性を、刺激・反応・意味・エピソードの“どのノードで読むか”で、B1/B3/B4/B5 が別々に測れるという点にある。初めての方は はじめに読む を先にどうぞ。

1

実験1と実験2の役割分担

実験1（済）実験2（これから）
問いグラフ自体がPTSDらしい連想を示すか（リアリティ）発話を加え、CAPS-5 の症状として再現できるか
測るものグラフ内部の活性量（般化勾配・侵入記憶）B1/B3/B4/B5 の挙動＋その回復・持続の動態＋患者発話
量(N)キュー100発話シミュレーションで多数のキュー×対話を回して確保
質の担保—井野先生の承認：①対応づけの臨床的妥当性を監査 ②サブサンプルを自ら採点し一致(κ)を検証

実験1は「地図の形が妥当」、実験2は「その地図から出る挙動が CAPS-5 の患者像に合う」を示す、という積み上げ。

2

鍵：同じ活性を、読む場所で症状が変わる

きっかけを入れると拡散活性が広がる。その活性をどの種類のノードで読むかで、対応する CAPS-5 の B 項目が変わる。

3

B項目ごとのグラフ測定

1つのキュー(発話)を入れて拡散活性を回すと、各ノードに活性が乗る。それを次の指数にまとめ、レベル L1〜L5 で平均して勾配を見る（＝過般化：薄い入力でも指数が立つか）。記号：R=反応活性、M=脅威・負の意味活性、E=負エピソード活性、S=感覚表象活性、C=文脈表象活性。

CAPS-5 B臨床が問うこと定量指数（グラフ上）必要機構
B5 生理反応 身体反応の有無・種類／きっかけ／回復まで 反応強度 R＝Σ生理反応活性（＝現行の怖がり度）／反応プロファイル＝発火した反応ノードの顔ぶれ（動悸・緊張…）／般化幅＝R の L4÷L1／回復＝Rが平静へ戻る減衰ターン数 既存＋残留活性
B4 心理的苦痛 きっかけでの動揺／どんなきっかけか／鎮静できるか 苦痛強度 M＝Σ脅威・負の意味活性（＝動揺の“中身”）／般化幅＝M の L4÷L1／回復＝Mの減衰ターン数（落ち着かせられるか） 既存＋残留活性
B1 侵入記憶 望まない想起／悩ませ度／他を考えられるか／月に何回 侵入強度 E＝Σ負エピソード活性／侵入率＝非トラウマ入力(L4)で E>θ の割合（＝望まない想起・頻度の代理）／制御困難＝集中度：E が全ノード活性に占める割合（高いほど他を考えにくい） 既存＋集中度指標
B3 フラッシュバック 再体験の強さ／現実認識の喪失（nowness）／どれくらい続くか 再体験強度 S＝感覚表象(S-rep)の活性／解離度＝S ÷ (S + C)（C=文脈表象。高いほど“今また起きてる”）／持続＝Sの減衰ターン数 S-rep/C-rep 分割（要追加）＋残留活性
B2 悪夢—夢の機構が無いため範囲外（DSM-5 は B 1つ以上で足りる）—
B1 と B3 の違い＋B3の理論的根拠（Brewin 二重表象理論）： B1「侵入記憶」は記憶が“浮かぶ”（あの出来事が勝手に頭に出る）＝負エピソードが光る、だけ。 B3「フラッシュバック」は、Brewin の理論では 感覚表象(S-rep：光景・音・身体反応)が強く発火するのに、 文脈表象(C-rep：“これは過去・あの場所の出来事”というタグ)が弱い＝解離として定義される（海馬の文脈化失敗に対応）。 だから B3 の核は「S が強いか」ではなく「S が強く、かつ C が弱いか」＝S/(S+C)。C がタグ付けに失敗するほど“今また起きている(nowness)”になる。
スコープ（正直な限界）：本モデルは S-rep/C-rep の不均衡（＝nowness の発生）までを扱う。 前頭前野の制御機構が絡む完全な解離（現実認識の完全な喪失）は範囲外と明記する。純粋な連想ネットワークでは 制御レイヤーを表現しないため。

B5・B4・B1 の“強度”(R/M/E)は実験1で既に測っている量。実験2は、そこに B3（S/C分割）・回復/持続の動態（残留活性）・発話・臨床承認を足す。

4

新しく必要な機構：残留活性の減衰（引き波）

B3 の「持続」、B4・B5 の「回復（自分を落ち着かせられるか）」、B1 の「他を考えられるか」は、いずれも時間とともに活性が引いていくかの問題。いまはターンごとに活性がリセットされるので、波が少しずつ引く仕組みを足す。

実装： A(t+1) = λ·A(t) + 新規入力。引き金の後に中立な発話を続け、活性が平静に戻るまでのターン数を測る。λ は全条件で固定し、回復の速さ（速い＝健常、遅い＝PTSD）は構造から創発させる（外付けにしない）。これ自体が「過覚醒が続く」という症状のモデル化なので、取ってつけた感なく本筋に乗る。
5

実験2の全体像：発話 × グラフ × 臨床承認

1つのキューに対して、次の3層を並べて評価する：

層内容
① グラフ測定上表の B1/B3/B4/B5 の量を拡散活性から算出（自動・大量）
② 患者発話その内部状態から患者の発話を生成し、CAPS-5 の B 挙動を自動評定（例 B3＝再体験的発話、B5＝身体反応の言及）
③ 臨床承認井野先生が「①↔②↔CAPS-5 の対応づけ」の妥当性を監査し、サブサンプルを自ら採点して一致(κ)を確認

①でN（量）を稼ぎ、③で質を担保。②が①と整合すれば「内部機構（グラフ）から CAPS-5 の症状発話が創発している」と言える。比較は忠実版グラフ vs 演技ベース vs 健常統制グラフ。

この設計の主張：PTSD の異なる症状（侵入記憶・フラッシュバック・心理的苦痛・生理反応）が、ひとつの記憶構造の異なる読み出しとして統一的に出てくる。症状ごとにルールを書くのではなく、構造から創発する——これが「演技」ではなく「構造から創発」の実証になる。
